LinkedIn Translator: Savage Truth

An AI-powered comedy translator that reveals the hidden emotional truth behind corporate LinkedIn posts.

Singapore Stupid Hackathon 2026

What is this?

Highlight any cringe LinkedIn post. A floating "Translate" button appears next to your selection. Click it. A new window opens showing the post's actual meaning — clichés underlined on the left, a Real Life Translation card on the right.

The real product is a Tampermonkey userscript that runs on linkedin.com. The widget below is a live demo so you can play with it without installing anything.

Live Demo

Requires the backend running locally on http://localhost:8000 — docker compose up from the project root.

Install the userscript

  1. Make sure the backend is running on http://localhost:8000 (this page wouldn't load otherwise).
  2. Install the Tampermonkey browser extension.
  3. Open userscript/linkedin-translator.user.js from the project repo — Tampermonkey will detect the metadata header and offer to install.
  4. Visit linkedin.com, highlight any post, click the floating Translate button.
